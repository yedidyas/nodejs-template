# nodejs-template
## Installation

- All the `code` required to get started

### Clone

- Clone this repo to your local machine using `https://github.com/yedidyas/nodejs-template.git`

### Setup

> install npm packages

```shell
$ npm install
```

### Run

> run from the terminal

```shell
$ npm start
```
### Browse

> browse from the browser

```shell
http://localhost:3000
```
